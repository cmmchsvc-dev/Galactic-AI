Galactic AI

Privacy Policy

Last updated: February 22, 2026

1. Overview

Galactic AI is a local-first, open-source AI automation platform. It is designed to run entirely on your own hardware. Your privacy is foundational to the project, not an afterthought.

2. Data Collection

Galactic AI does not collect, store, transmit, or sell any user data. There is no telemetry, no analytics, no tracking, and no remote logging of any kind.

Specifically, Galactic AI does not:

  • Send any data to Galactic AI servers (there are none)
  • Track your usage, conversations, or behavior
  • Store cookies or browser fingerprints
  • Collect personal information, email addresses, or IP addresses
  • Include any analytics or tracking scripts on this website

3. Local Data Storage

All data generated by Galactic AI (conversations, memory files, logs, configuration) is stored locally on your machine in the installation directory. This data never leaves your computer unless you choose to send it somewhere.

4. Third-Party API Providers

Galactic AI connects to third-party AI providers (such as Google, Anthropic, OpenAI, NVIDIA, Groq, xAI, OpenRouter, and others) using your own API keys. When you send a message to an AI model, the content of that message is transmitted to the selected provider.

Each provider has its own privacy policy and data handling practices. Galactic AI does not control, process, or store any data handled by these third-party providers. You are responsible for reviewing and accepting the privacy policies of any providers you choose to use.

5. Telegram, WhatsApp, and Other Integrations

If you connect Galactic AI to messaging platforms (Telegram, WhatsApp, etc.), messages are routed through those platforms' APIs. Galactic AI processes messages locally on your machine but the transport layer is governed by the respective platform's privacy policy.

6. X (Twitter) API Integration

Galactic AI can optionally post to X (Twitter) on your behalf using the X API with your own OAuth credentials. When you use this feature:

  • Your OAuth tokens are stored locally in your configuration file
  • Post content is sent directly to X's API from your machine
  • Galactic AI does not store, log, or analyze your X data beyond the local session

7. Website

This website (cmmchsvc.net) is a static site hosted on GitHub Pages. It does not use cookies, analytics, or tracking scripts of any kind. GitHub Pages may collect standard server logs as described in GitHub's Privacy Statement.

8. Open Source Transparency

Galactic AI is fully open source under the MIT License. You can inspect every line of code to verify these privacy claims at github.com/cmmchsvc-dev/Galactic-AI.

9. Children's Privacy

Galactic AI is not directed at children under 13. We do not knowingly collect data from anyone, including children.

10. Changes to This Policy

If this privacy policy is updated, the changes will be posted on this page with an updated date. Since Galactic AI collects no data, changes are expected to be minimal.

11. Contact

If you have any questions about this privacy policy, you can reach out via GitHub Issues.
