## Reporting a Vulnerability

We take the security of Galactic AI seriously. If you believe you have found a security vulnerability, please report it to us responsibly.

**Please do not open a public GitHub issue for security vulnerabilities.**

Instead, please send a detailed report to:
**chesley@cmmchsvc.net**

### What to include:
- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact if exploited.
- Any suggested fixes or mitigations.

We will acknowledge your report within 48 hours and provide a timeline for a fix if necessary. Thank you for helping keep Galactic AI secure!
