# v1.1.2 Release — Design Document

**Date:** 2026-02-23
**Version bump:** 1.1.1 → 1.1.2
**Theme:** Skills Ecosystem

---

## Overview

v1.1.2 introduces the Skills ecosystem — a full architectural evolution of the plugin system into a first-class, self-authoring, community-extensible skill platform. All 6 core plugins have been migrated to `GalacticSkill` subclasses, Byte can write and load new skills at runtime, and the Control Deck now shows a rich Skills tab.

This release document covers all doc/website/package updates needed to ship v1.1.2 publicly.
